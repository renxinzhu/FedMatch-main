# FedMatch in pytorch

### How to start training

1. Create a virtual environemnt and install dependencies

   ```shell
   python3 -m venv env
   source env/bin/activate
   python3 -m pip3 install -r requirements.txt
   ```

2. Change hyperparameters in `train.py`

3. Start training

   ```shell
   python3 train.py
   ```
